Vinculacion de una base de datos en una pagina php se puede realizar de manera sencilla teniendo en cuenta 4 datos importante: Localhost: indica a donde se conecta con la DB. Si la conexion es en el mismo servidor, este es el valor que debe ir. Si la conexion se da en otro servidor, debemos incluir la Ip del servidor. En este ultimo caso, el servidor debera tener las conexiones remotas abiertas, Usuario: Es el usuario de la base de datos. En general lleva el siguiente formato usuarioredgourmet_usuariodb Clave: Es la clave que se corresponde con el usuario "usuarioreunad_usuariodb". Nombre de la Base de datos “redgourmet” La vinculacion de la BD a la página es muy importante ya que sin esta no se podria almacenar la informacion cuando se tienen formularios de registro, historiales e inventarios. Adicionalmente es indispensable porque es siempre importante garantizar que la pagina cumpla con un correcto funcionamiento, siendo la BD un elemento de total importancia para el manejo de informacion y conexion entre diferentes enlaces que tiene una pagina.
